# HuiTools 模块

## 概述

> HuiTools 是一个基于 Vue3+ElementPlus 封装的模块化实用工具库。

## 使用

> 1. 通过 npm 安装 hui-ui 组件库[详见组件安装指引](../../README.md)
> 2. 在需要使用方法的组件中引入 HuiTools 模块并使用其中的方法

```js
import { HuiTools } from 'hui-ui'
```
